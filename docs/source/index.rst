PreppyData Documentation
========================

Welcome to the PreppyData home page!

PreppyData is a web-based platform designed to simplify data preprocessing for various datasets. Our goal is to provide users with customizable options for encoding, outlier detection, feature selection, and more, making data preparation accessible to everyone.
